# CareerHub

개발자 취업 준비 과정을 통합 관리하는 Career Management Platform

## 프로젝트 소개

CareerHub는 취업 준비생이 학습 기록, 프로젝트 경험, 지원 현황을 한 곳에서 관리할 수 있는 웹 서비스입니다.

사용자는 자신의 성장 과정을 데이터로 기록하고, AI 분석을 통해 부족한 역량과 개선 방향을 확인할 수 있습니다.

## 주요 기능

- 회원가입 / 로그인
- 학습 기록 관리
- 프로젝트 관리
- 지원 현황 관리
- 이력서 업로드
- 통계 대시보드
- AI 커리어 분석
